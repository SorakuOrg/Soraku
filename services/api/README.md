# @soraku/api — Central API
Central API untuk seluruh platform Soraku. Semua client apps (web, stream, mobile, bot) berkomunikasi melalui API ini.
Tanggung jawab: Auth, users, community, content, events, gallery, supporters, streaming, notifications, webhooks.
Stack (planned): Hono / Next.js API routes + Drizzle ORM + Supabase + Zod
